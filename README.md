# dt.styles

An assorted collection of styles for darktable. Set up in a hierarchical way. Depending on the view you are in the styles are represented in a double or a single expandable layer:

lighttable view:
```
     Colour
         Agfa
             Vista 200
```

darkroom view:
```
     Colour
         Agfa|Vista 200
```
